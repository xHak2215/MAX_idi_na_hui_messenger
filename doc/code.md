# структура проекта

**директории:**

- `/doc` - документация кроме `README.md` тут
- `/libs` - библиотеки (не считая сторонних) и дополнительные программы тут
- `/test` - тут тесты 
- `/media` - медиа данные тут
- `/chat` - тут хранятся чаты 

**файлы:**

- `/requirements.txt` - список необходимых библиотек 
- `/data_bese.py` - база данных 
- `/api.py` - API для взаимодействие с БД (скорее всего главный API)
- `/test/api_test_user.py` - тест API(`/api.py`) базы данных, все связанное с пользователями
- `/libs/ffmpeg_tool.py` - древний СШК для конвертации фото, видео и извлечения звука с помощью утилиты ffmpeg
- `/libs/gmail_send.py` - отправка письма на почту 
- `/test/api_test_chat.py` -  тест API(`/api.py`) базы данных, все связанное с чатами
- `/test/api_test_media.py` - тест API(`/api.py`) загрузка и отправка медиа
- `/libs/zip_images.py` - сжатие изображения но указанное количество %
- `/media/media_config.json` - хранит последние значение id медиа(`media_id`), нужен для создания id новому файлу 
- `/libs/token_creat.py` - функция для создания уникального токена пользователя
- `/libs/messages_menager.py` - отвечает за работу с чатами и сообщениями в них
- `/web_ui_server.py` - сервер поднимающий сайт
